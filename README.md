## Tema
Avaliação de desempenho de comunicação cliente-servidor em jogos online utilizando TCP, com simulação baseada no Minecraft.

## Introdução
A comunicação cliente-servidor é fundamental em jogos online, permitindo a troca de dados entre jogadores e servidor em tempo real. Em jogos como Minecraft, esse modelo é essencial para sincronizar ações e manter a consistência do ambiente.

## Proposta
Este projeto implementa uma simulação cliente-servidor em Python, usando sockets e TCP, para avaliar latência e tempo de resposta em uma comunicação inspirada em jogos online.

## Tecnologias
- Python
- Biblioteca `socket`
- Protocolo TCP
